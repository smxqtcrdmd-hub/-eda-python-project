# Exploratory Data Analysis using Python

This project demonstrates basic Exploratory Data Analysis (EDA) using Python.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib

## Project Description
The project analyzes a simple sales dataset to understand trends and summary statistics.
Basic visualizations such as line plots and bar charts are used to represent sales and profit.

## Learning Outcome
- Data handling using Pandas
- Statistical summary of data
- Basic data visualization
- Understanding EDA workflow

This project is created as part of self-learning in Data Science.
